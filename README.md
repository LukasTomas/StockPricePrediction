# Stock Price Movement Direction Prediction

## Project description

Use sequence model (such as LSTM, GRU, ...) to predict stock price movement direction (up/down) and its probability based on historical data on the U.S. stock market.

## Pokyny k dotažení potřebných velkých dat, jsouli mimo git repo


Stručné shrnutí zadání (několik vět);
Použijte sekvenční model (například LSTM, GRU, ...) k předpovědi směru pohybu ceny akcií (nahoru/dolů) a jeho pravděpodobnosti na základě historických dat o akciovém trhu USA. Konkrétně se bude jednat o OHLC data stažená z yahoo finance.
